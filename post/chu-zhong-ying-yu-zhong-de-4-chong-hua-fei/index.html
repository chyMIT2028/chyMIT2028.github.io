初中英语中的4种“花费”

2023-11-08

take

用法：

it takes sb. some time to do sth.

只能表示时间的“花费”。

spend

用法：

it spends sb. some money on sth.

it spends sb some time on sth./in doing sth.

即可表示时间的“花费”，也可表示金钱的“花费”。

cost

用法：

sth. cost some money

只能表示金钱的“花费”，并且主语只能为物。

pay

用法：

sb. pay some money for sth.

只能表示金钱的“花费”。
